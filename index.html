Files

ls

Lists your files.

ls -l

Lists your files in 'long format', showing additional details.

ls -a

Lists all files, including hidden files.

more filename

Shows the first part of a file.

emacs filename

Edits a file using emacs.

mv filename1 filename2

Moves or renames a file.

cp filename1 filename2

Copies a file.

rm filename

Removes a file.

Directories

mkdir dirname

Creates a new directory.

cd dirname

Changes the current directory.

pwd

Prints the current directory.

Processes

ps

Displays information about active processes.

top

Displays a dynamic real-time view of system processes.

kill PID

Terminates a process by its ID.

Network Communication

ping hostname

Tests connectivity with another host.

traceroute hostname

Traces the route packets take to a destination.

ssh username@hostname

Provides secure remote access to a system.

Text Editors

vim filename

Edits a file using Vim.

emacs filename

Edits a file using Emacs.

nano filename

Edits a file using Nano.
